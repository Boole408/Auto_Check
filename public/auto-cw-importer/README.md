# Auto_CW 导入助手

## 安装

1. 下载并解压 `auto-cw-importer-extension.zip`。
2. 打开 Chrome 或 Edge 的扩展管理页。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择解压后的 `auto-cw-importer` 文件夹。

如果已经安装过旧版，请先删除旧的“Auto_CW 导入助手”，再加载新版文件夹，或在扩展管理页点击“重新加载”。

## 使用

1. 先打开 Auto_CW 并登录一次。
2. 在同一个浏览器里打开 MUYUAN 或其它已配置站点，完成网页登录。
3. 点击浏览器工具栏里的“Auto_CW 导入助手”。
4. 确认 Auto_CW 地址后点击“导入当前账号”。

扩展会读取当前站点的 Cookie，并调用 Auto_CW 的导入接口。它不会保存你的 LinuxDo 密码。
如果站点不使用 Cookie，扩展会继续尝试读取当前页面的 localStorage/sessionStorage 登录态。
